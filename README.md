## 🚴 Usage


### 💾 Download the respository onto your computer with `git clone`

```
git clone https://github.com/ECE421Group11/Project3Rust.git
cd Project3Rust
```


### 🛠️ Build with `cargo build` and `wasm-pack build`

```
cargo build
wasm-pack build
```

### 🏃‍♀️ Run with `npm run start`

```
cd www
npm install
npm run start
```
